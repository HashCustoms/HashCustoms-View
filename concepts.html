Project Morse Code

Logical Brief

Read here for all your doubts

As we know it the project morse code is based on the morse code. Which was used in wars earlier with idea of converting every number or alphabet into dots and dashes so that the message can be conveyed in the form of long and short taps. Thereby the project morse code will be based on a unique number that will be assigned to every client and the same message will be conveyed over the model of the client in the expression of stripes or the pattern of light which can be recognised giving the motorcycle a unique identity. There will be only 27 motorcycles rhat will be converted into morse code as a tribute to the inventor of the code, Samuel Morse, an american painter and inventor (born on 27th April 1791).
Keeping the radicality in mind the concept has been evolved reflecting minimalism as well, because thats what the morse code conveys with the ideas of simplifying every sentence and word into dots and dashes.
